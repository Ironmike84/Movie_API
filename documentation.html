MOVIE API ============================================================================================================================================= The Movie API Call creates a server Request and imports the HTTP module and returns the URL Module Index File. If user adds documentation to localHost:8080/documentation the user will be handed the documentation file. The Movie API will: - Search and find Details about movies (such as Titles directors, genres.) - Users can create thier own account add personal information - Create a list of thier favorite - Users Can Delete Movies Also from their list ==============================================================================================================================================
